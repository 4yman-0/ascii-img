# ascii-img
Convert images to ASCII

## How to install
1. Clone the repository:
```
git clone https://github.com/4yman-0/ascii-img
```

2. Choose a front-end (for example: `ascii-img-cli`):
```
cd ascii-img-cli
```

3. Build with [`cargo`](https://www.rust-lang.org/):
```
cargo build
```
